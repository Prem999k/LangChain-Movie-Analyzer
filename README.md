# 🤖 Generative AI Projects Collection

A collection of Generative AI applications built using:

- LangChain
- Streamlit
- Mistral AI
- Groq
- OpenAI
- Hugging Face
- Pydantic
- Embedding Models

This repository contains multiple AI-powered applications including chatbots, structured output systems, movie analyzers, and embedding-based projects.

---

# 🚀 Projects Included

## 🎬 Movie Analyzer AI

Extracts movie information from movie paragraphs using LLMs and Structured Output Parsing.

### Features

- Movie Title Extraction
- Director Detection
- Genre Identification
- Cast Extraction
- IMDb Rating Extraction
- AI Summary Generation
- JSON Structured Output

---

## 💬 AI Mood ChatBot

### Modes

- 😂 Funny Mode
- 😡 Angry Mode
- 😢 Sad Mode

### Features

- Real-time AI Chat
- Mood Switching
- Chat History
- Streamlit Chat UI
- Mistral AI Integration

## 🧠 Hugging Face Embedding Models
